| Key | Output | | Key | Output |
|-----|--------|-|-----|--------|
| Q | → line end (Cmd+→) | | Y | Home |
| W | `\|` | | U | `{` |
| E | `(` | | I | `}` |
| R | `)` | | O | Page Dn |
| T | Page Up | | P | End |

| F | ↑ | | L | `-` (hyphen) |
| G | `<` | | `;` | `–` (en dash) |
| | | | `'` | `—` (em dash) |
| Z | ←Desktop | | N | Mute |

**Number row:** 1–0 = F1–F10, `-` = F11, `=` = F12

**Tap Caps Lock alone** = Escape
**Left Shift + Caps Lock** = real Caps Lock toggle

## Layer 3: Tab (tap=Tab, hold=Layer 3)

### Tier 1 — Direct Cmd shortcuts (2 steps: hold Tab + press)

| Key | Output |
|-----|--------|
| C | Cmd+C (Copy) |
| V | Cmd+V (Paste) |
| X | Cmd+X (Cut) |
| S | Cmd+S (Save) |
| F | Cmd+F (Find) |
| T | Cmd+T (New tab) |
| W | Cmd+W (Close tab) |
| U | Cmd+Z (Undo) |
| R | Cmd+Shift+Z (Redo) |
| Space | Cmd+Space (Spotlight) |
| A | Cmd+Tab (App switcher) |
| Return | Cmd+Return (Wispr Flow) |

### Tier 2 — One-shot modifiers, right hand (3 steps: hold Tab + tap → release Tab → tap target)

| Key | Modifier |
|-----|---------|
| J | ⌘ Cmd |
| K | ⌃ Ctrl |
| L | ⌥ Alt/Option |
| `;` | ⇧ Shift |
| H | ⌘⇧ Cmd+Shift (both armed atomically — use this instead of stacking J+`;`) |
| N | ⌘⌥ Cmd+Option (both armed atomically) |
| M | ⌃⌘ Ctrl+Cmd (both armed atomically) |

**Capitalize:** hold Tab → tap `;` (arms Shift) → release Tab → type letter

## Examples

### Layer 2 (Hold Caps Lock)

| What you want | Keystrokes |
|---------------|------------|
| Go to start of line (no selection) | Hold Caps Lock → A |
| Go to end of line (no selection) | Hold Caps Lock → Q |
| Move cursor left | Hold Caps Lock → D |
| Move cursor up | Hold Caps Lock → F |
| Jump to end of line | Hold Caps Lock → P |
| Jump to start of line | Hold Caps Lock → Y |
| Page down | Hold Caps Lock → O |
| Select to end of line | Hold Caps Lock → Return |
| Select word right | Hold Caps Lock → `/` |
| Select word left | Hold Caps Lock → `.` |
| Select to line start | Hold Caps Lock → `,` |
| Type `[` | Hold Caps Lock → S |
| Type `{` | Hold Caps Lock → U |
| Type `(` | Hold Caps Lock → E |
| Type en dash `–` | Hold Caps Lock → `;` |
| Type em dash `—` | Hold Caps Lock → `'` |
| Switch desktop left | Hold Caps Lock → Z |
| Switch desktop right | Hold Caps Lock → X |
| Escape | Tap Caps Lock alone |

### Layer 3 Tier 1 (Hold Tab — direct shortcuts)

| What you want | Keystrokes |
|---------------|------------|
| Copy | Hold Tab → C |
| Paste | Hold Tab → V |
| Cut | Hold Tab → X |
| Save | Hold Tab → S |
| Undo | Hold Tab → U |
| Redo | Hold Tab → R |
| Find | Hold Tab → F |
| New tab | Hold Tab → T |
| Close tab | Hold Tab → W |
| Open Spotlight | Hold Tab → Space |
| Switch app (app switcher) | Hold Tab → A |
| Toggle Wispr Flow | Hold Tab → Return |

### Layer 3 Tier 2 (Hold Tab — one-shot modifiers)

| What you want | Keystrokes |
|---------------|------------|
| Cmd + any key | Hold Tab → tap J → release Tab → key |
| Ctrl + any key | Hold Tab → tap K → release Tab → key |
| Option + any key | Hold Tab → tap L → release Tab → key |
| Capitalize a letter | Hold Tab → tap `;` → release Tab → letter |
| Cmd+Shift + any key | Hold Tab → tap H → release Tab → key |
| Cmd+Shift+P (VS Code palette) | Hold Tab → H → release Tab → P |
| Cmd+Shift+K (delete line VS Code) | Hold Tab → H → release Tab → K |
| Cmd+Shift+F (global find) | Hold Tab → H → release Tab → F |
| Ctrl+C (terminal kill) | Hold Tab → tap K → release Tab → C |
| Cmd+Option + any key | Hold Tab → tap N → release Tab → key |
| Cmd+Option+I (DevTools) | Hold Tab → N → release Tab → I |
| Ctrl+Cmd + any key | Hold Tab → tap M → release Tab → key |
| Ctrl+Cmd+F (fullscreen) | Hold Tab → M → release Tab → F |

### Tab key

| What you want | Keystrokes |
|---------------|------------|
| Tab character | Tap Tab alone (or S + D simultaneously as backup) |

### Combos (base layer)

| What you want | Keystrokes |
|---------------|------------|
| Backspace | D + F simultaneously |
| Escape | F + G simultaneously |
| Space | J + K simultaneously |
| Enter | K + L simultaneously |
